## 🛠️ How to Use

### Opening the Website
1. Open `index.html` in your web browser
2. Navigate using the menu at the top
3. Click on service cards, client names, or CTA buttons to explore

### Customization
To customize the website:

#### Update Company Information
Edit header contact details in each HTML file:
```html
<span>📞 +91 9545551435 | +91 9545551444 | +91 9850842287</span>
<span>📧 msquarehr@gmail.com</span>
```

#### Change Colors
Modify CSS variables in `css/styles.css`:
```css
:root {
    --primary-blue: #003d82;
    --primary-green: #2ecc71;
    /* Update other colors as needed */
}
```

#### Add Images
1. Create an `assets` folder
2. Add images to the folder
3. Reference them in HTML: `<img src="assets/image-name.jpg" alt="Description">`

#### Update Content
Simply edit the HTML files with your text editor and update the content sections.
